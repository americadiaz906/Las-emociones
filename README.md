# Las-emociones
Una experiencia interactiva diseñada para que los alumnos de preescolar identifiquen y expresen emociones a través de situaciones cotidianas. Este juego promueve la inteligencia emocional y la representación física, ayudando a los pequeños a reconocer alegría, tristeza, sorpresa, miedo y enojo de forma lúdica y sencilla.
